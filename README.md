# Bootcamp-MB3

Pruebas SQL Y PYTHON para consultas de dataframes, extraemos los datos de diversas fuentes ( web scraping, apis, bases de datos, archivos locales como json, csv o xml ) y lo que hacemos es llevarlos a un formato entendible y facil de analzar como es un dataframe, en este repo encontramos una notebook con consulta sql y pandas para un mismo dataframe, un archivo .py con el proceso y un pipeline con las funciones de cada paso, para en el futuro poder reutilizar y automatizar
